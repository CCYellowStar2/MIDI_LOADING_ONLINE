# MIDI_LOADING_ONLINE

 <p ling="center">
  在線式的MIDI轉換器


> [!tip]
>
> 本項目目前為早期開發階段，有任何問題歡迎提交Pull requests 與 Issue。


>[!worrior]
>
> 本項目使用 << GPL-3.0 license >> 作為版權宣告。

## 功能
- 目前支持將文件由 .xml .mxl 與 mdi 轉換在線實時轉譯，並且內建詞彙庫，用戶可以自行更改按鍵設置。
- 非開發用戶請下載 Releases 版本
